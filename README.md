# Flutter Labs assignments

This repository contains 4 lab assignments for the "Exploring Common Flutter Widgets" course.

## Project Structure
- `profile_card_lab/`: Lab 1 - Profile Card UI
- `bottom_nav_lab/`: Lab 2 - Bottom Navigation and Screens
- `catalog_lab/`: Lab 3 - Product Catalog (Grid View)
- `registration_lab/`: Lab 4 - Registration Form with Validation

## How to Run
To run any of the labs, navigate into the project folder and use the flutter run command:
```bash
cd <lab_folder_name>
flutter run
```

## Requirements for Submission
- Screenshots of the running apps.
- Widget tree for each app (provided in the `walkthrough.md`).
- Push this entire repository to your GitHub.
